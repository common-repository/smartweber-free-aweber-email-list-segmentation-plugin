== Description ==

The Ultimate Aweber Wordpress Plugin

SmartWeber extends the functionality of your Aweber account by allowing you to segment your lists easily and quickly.

How it works

1. You create a link in your admin panel
2. Choose whether you want to copy or move subscribers to a new list
3. Once a subscriber clicks a link he/she is copied or moved to the new list.

== Installation ==


1. Upload and extract `smartweber.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Click on `SmartWeber` in your dashboard to start segmenting your lists

== Frequently Asked Questions ==
